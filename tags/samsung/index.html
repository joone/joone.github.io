SAMSUNG Windows Mobile SDK 1.0 Released!
Samsung Windows Mobile SDK 1.0 was released! You can download Samsung Windows Mobile SDK 1.0 from SAMSUNG Mobile Innovator web site, but it needs to be registered for downloading. Of course, it’s free. It consists of API Spec, programming guide, installation guide, header files & library for device APIs, and sample code. The APIs is used for providing advanced features unique to Samsung phones. You might already have experience using advanced capabilities featured in the Windows Mobile APIs, like GPS, Bluetooth, or multimedia support. More
